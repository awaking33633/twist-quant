
# Twist - 缠论技术分析库

识别趋势 跟随趋势 而非预测趋势

* 聚焦于多级别K线形态分析 5min、15min、30min、60min、日线、周线等级别
* 挖掘缠论一买、二买等关键买点
* 低吸操作体系: 买在下跌 买在背驰

## K线结构分析

1. 遍历K线 分析寻找顶底分型位置
2. 根据顶底分型位置 判断上涨、下跌结构形成 (即缠论中的笔)
3. 根据上涨笔、下跌笔 判断中枢结构形成
4. 根据笔、中枢 判断趋势结构

## 备注

* K线应该从后往前看 以清晰解构历史行情 明确当前行情走势结构
